Οποιο αρχειο μπει μεσα στο φακελο C:\Home Portal\xampp\htdocs\uploads θα φαινεται στο κατω δεξια κουτακι στον πινακα ανακοινωσεων.
Αυτο το εγγραφο readme μπορει να αφαιρεθει αν το διαγραψετε απο το φακελο.

μεγαλυτερα αρχεια μπορειτε να στελνετε αρχεια Torrent.
